This repo examined/employed five different Deep Learning architectures on the task of classifing music genres. There are ten classes, e.g. Jazz, Blues, classic, and etc. Detailed by the report.

Experiments are carried out in Google Colab. Codes are provided in jupyter notebooks. Weight are provided.

There are five notebooks, which map to five different architectures and can be run seperately.

Dataset used is the famous GTZAN Dataset, https://www.kaggle.com/datasets/andradaolteanu/gtzan-dataset-music-genre-classification

The weights of VGG16 is uploaded to google drive, https://drive.google.com/file/d/15brWuNcM04Id3LeNuKBywj5WBCTPgqJB/view?usp=sharing
